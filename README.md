# twitter-bot-v2
Twitter-Bot-GPT  (Node.js y OpenAI)

### Install

```
$ npm install
```
Add the .env file and complete Twitter and OpenAI Kyes APIs
```

#dev
APP_TOKEN=""

#user
APP_KEY=""
APP_SECRET=""
ACCESS_TOKEN=""
ACCESS_SECRET=""

#openai
OPENAI_API_KEY=""

```

### Run

```
$ node index.js
```
