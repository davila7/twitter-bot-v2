# twitter-bot-v2
Bot en Twitter con Node.js y OpenAI

### Install

```
$ npm install
```
Agrega el archivo .env y agrega las API Kyes de Twitter y OpenAI
```

#dev
APP_TOKEN=""

#user
APP_KEY=""
APP_SECRET=""
ACCESS_TOKEN=""
ACCESS_SECRET=""

#openai
OPENAI_API_KEY=""

```

### Run

```
$ node index.js
```
